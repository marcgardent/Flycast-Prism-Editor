# Flycast Prism Editor

A lightweight, cross-platform standalone viewer designed to inspect EXR files generated by the Flycast Prism / deferred processing engine.

## Overview

This tool provides a streamlined way to visualize EXR output from Flycast.

## Technical Stack

- **UI Framework:** [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) (Modern, dark-themed UI)
- **Image Processing:** [OpenImageIO](https://github.com/OpenImageIO/oiio) / [PyOpenEXR](https://github.com/jamesbowman/PyOpenEXR)
- **Environment Management:** Conda
- **Packaging:** [Briefcase (BeeWare)](https://beeware.org/project/projects/tools/briefcase/)
- **Target Formats:** - **Linux:** AppImage (Standalone)
  - **Windows:** MSI Installer / Portable EXE

## Prerequisites

- **Conda** (Anaconda or Miniconda)
- **Python 3.10+**
- For Linux/windows packaging: `docker`
