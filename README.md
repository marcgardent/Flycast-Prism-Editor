
# Flycast GBuffer Viewer

A lightweight, cross-platform standalone viewer designed to inspect EXR files generated by the Flycast experimental GBuffer / deferred processing engine.

## Overview

This tool provides a streamlined way to visualize high-dynamic-range (HDR) EXR output from Flycast. It is built with performance and portability in mind, using a modern Python stack to ensure compatibility across Linux and Windows.

## Technical Stack

- **UI Framework:** [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) (Modern, dark-themed UI)
- **Image Processing:** [OpenImageIO](https://github.com/OpenImageIO/oiio) / [PyOpenEXR](https://github.com/jamesbowman/PyOpenEXR)
- **Environment Management:** Conda
- **Packaging:** [Briefcase (BeeWare)](https://beeware.org/project/projects/tools/briefcase/)
- **Target Formats:** - **Linux:** AppImage (Standalone)
  - **Windows:** MSI Installer / Portable EXE

## Prerequisites

- **Conda** (Anaconda or Miniconda)
- **Python 3.10+**
- For Linux packaging: `docker` or development tools (system headers for EXR libraries)
